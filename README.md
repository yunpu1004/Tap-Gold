- **게임명:** Tap & Gold
- **개발기간:** 한 달
- **개발인원:** 한 명 (개인)
- **장르:** 클리커게임
- **플랫폼:** 플레이스토어

## 프로젝트 소개
이 프로젝트는 화면을 탭하여 얻은 골드를 사용해 각종 업그레이드를 하여 더 많은 골드를 얻고, 환생을 통해 얻은 재화를 사용해서 스킬 레벨업을 하거나 특수한 효과를 가진 아티팩트를 수집하는 게임입니다. 게임 플레이어는 이러한 방식으로 기하급수적인 성장을 경험할 수 있습니다.
